# Portfolio Project

This repository contains a simple personal portfolio website that showcases projects, skills, and contact information. It is built using basic web technologies and is intended as a starting point for demonstrating frontend development skills.

## 📁 Files

- `main.html` – The main HTML page for the portfolio.
- `style.css` – Stylesheet used by the HTML page to define the look and feel.

## 💡 Features

- Responsive layout (mobile-friendly).
- Sections for about, projects, and contact.
- Custom styling using CSS.

## 🚀 Getting Started

1. Clone or download this repository.
2. Open `main.html` in your web browser (double-click or use `File → Open`).

No build tools or servers are required – just a modern web browser.

## 🛠️ Technologies

- HTML5
- CSS3

## 📌 Notes

Feel free to modify the content and styles to personalize the portfolio. Add new sections, images, or links to actual projects as needed.
